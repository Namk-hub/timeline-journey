# Namrat.dev

An interactive timeline website that visually showcases my developer journey through different phases, experiences, and milestones. Built with a modern UI, smooth animations, and storytelling elements to make the journey feel immersive and personal instead of a traditional resume.

## 🚀 Tech Stack

### Core
- **React** for building the UI and reusable components  
- **Vite** for fast development and build setup  

### Animations
- **GSAP** for scroll animations and cinematic transitions  
- **Framer Motion** for smooth interactive motion effects  

### 3D & Graphics
- **Spline** for interactive 3D assets  
- **Three.js** for rendering 3D elements on the web  

### Styling & Utilities
- **Vanilla CSS** for custom styling and layouts  
- **React Bits** for extra animated UI components and effects
